
# ☕ Coffee Shop Sales Analysis using SQL

## 📌 Project Overview

This project analyzes a Coffee Shop Sales dataset using MySQL to answer real-world business questions and generate actionable business insights.

The analysis focuses on revenue, sales performance, product performance, store performance, and customer purchasing trends.

---

## 🎯 Project Objectives

- Analyze total revenue generated
- Identify top-selling products
- Compare store performance
- Analyze monthly and hourly sales trends
- Generate business insights using SQL
